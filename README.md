# Galen Example #

## Galen Syntax ##
 > ### How to install Galen ###
 > 
 > npm install -g galenframework-cli
 
 > ### How to execute tests ###
 >
 > cd galentests
 > galen test localhost.test --htmlreport reportdir

## Java Syntax ##

 > ### How to execute tests: ###
 >
 > cd javatests
 > mvn test
